# Form

Simple form wrapper to build Lesli forms


## Props and emited functions

**@submit()** 
Emited when form is submited


### examples

```html
<lesli-form class="mb-4" @submit="isEditable ? onUpdate() : onCreate()">
</lesli-form>
```

```raw
<lesli-form class="mb-4" @submit="isEditable ? onUpdate() : onCreate()">
</lesli-form>
```
