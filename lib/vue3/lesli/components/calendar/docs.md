# Calendar
Provide a set of html view components to allow user work with date and time

## Props single date
- v-model: (Date) v-model of the date picker

## Props date ranges
- v-model: (object) v-model of the date picker

## Use examples:

```html
<lesli-calendar :model="mydate">
</lesli-calendar>
```


```raw
<lesli-calendar :model="mydate">
</lesli-calendar>
```
